# mdeditor
shitty markdown editor

## requirements
+ node installed on your system
+ cmake, c++ compiler

## build/run

### Linux
    git clone https://github.com/unkomogu2/mdeditor.git
    cd mdeditor
    ./build-run.sh

### Windows
    git clone https://github.com/unkomogu2/mdeditor.git
    cd mdeditor
    ./build-run.bat
